Q - 9. How can one create an alert in Bootstrap?

Ans.
Bootstrap provides a set of alert classes that can be used to display various types of alerts, such as success messages, warning messages, error messages, and informational messages.

To create an alert, you can use the following classes:

  • .alert-success: For success messages.
  • .alert-warning: For warning messages.
  • .alert-danger: For error messages.
  • .alert-info: For informational messages.
