Q - 4. What do you know about the Bootstrap Grid System?

Ans.
The Bootstrap Grid System is a responsive, mobile-first grid system that allows developers to create layouts with up to 12 columns across various screen sizes.

Key points about the Bootstrap Grid System:

  • Responsive Design: Automatically adjusts layout based on screen size and device.
  • 12-Column Layout: Provides flexibility with a grid system divided into 12 columns.
  • Column Classes: Predefined classes like .col-, .col-sm-, .col-md-, etc., define column widths at different breakpoints.
  • Grid Nesting: Allows nesting rows and columns for creating complex layouts.
  • Container: Grid layouts are typically wrapped in .container or .container-fluid for fixed-width or full-width containers respectively.
